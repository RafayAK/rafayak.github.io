# rafayak.github.com

Personal site for Rafay — AI Engineer. Portfolio + blog, built with Astro.

Warm letterpress aesthetic (Lora, cream canvas, warm dark mode), a markdown blog
with animated squiggly dividers and a pencil reading-rail, hover link previews,
and an ambient music toggle.

## Develop

```bash
npm install
npm run dev      # http://localhost:4321
npm run dev:logs # follow live dev server logs
npm run dev:status
npm run dev:stop
npm run build    # → dist/
npm run preview
```

## Docs

- `AGENTS.md` — stack, conventions, and feature notes for coding agents.
- `DESIGN.md` — colors, type, and component specs.

## Deploy

Pushed to `main` → GitHub Actions builds and deploys to GitHub Pages.
